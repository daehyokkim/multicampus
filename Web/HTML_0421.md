### HTML5

- 모든사람들이 웹을 사용할 수 있도록 기능을 구성해야 되기 때문에  웹 표준안을 발표
- 동적인 내용을 처리하도록 발표
- 정보를 보기 쉽게 **편집해서** 보여주는 기술

주요기능

- 장치접근
  - 장치에 접근해 정보,기능 사용할 수있다
  - ex) 배터리 잔량,위치,알람,스마트폰등 접근 가능
- 시맨틱
  - 정보를 의미상으로 접근하여 정보 분석,자료를 검색 및 처리해서 제공

태그와 요소

- 표기법 `<요소 이름>내용</요소 이름>`

- 요소 : HTML 페이지를 구성하는 각 부품
- 태그 : 요소를 만들 떄 사용하는 작성 법



속성

- 태그에 추가 정보를 부여할 때 사용하는 것
- `< 요소 이름 속성1="">내용</요소 이름>`

## 태그

#### 앵커 태그

- `<a href =" "...> </a>` 사용

- 속성의 id 지칭

  ```html
  <a href="#id_id">Alpha</a>
  <h1 id="h1_id">
      a 태그 클릭시 여기로 이동~~
  </h1>
  ```

  

- URL 지칭

  ```html
  <a href="go.html" action="get or post">해당 url 이동</a>
  <!-- go.html-->
  <h1>go.html을 열기~~</h1>
  ```

#### 목록 태그

- 순서가 없는 목록

  ```html
  <ul>
      <li>하나</li>
      <li>둘</li>
  </ul>
  <!-- 출력 -->
  •하나
  •둘
  ```

- 순서가 있는 목록<ol>
      <li>하나</li>
      <li>둘</li>
  </ol>
  <!-- 출력 -->
  1.하나
  
  2.둘

#### 테이블 태그

- 테이블 생성

  - `<thead>` : 테이블 표의 제목 구간
  - `<tbody>` : 테이블 표의 데이터 입력 구간
  - `<tr>`  : 표의 행 삽입
  - `<th>`  : 표의 제목 셀 생성
  - `<td>`  : 표의 일반 셀 생성

  ```html
  <table>
      <thead>
          <tr>
              <th>제목1</th>
          </tr>
      </thead>
      <tbody>
          <tr>
          	<td>내용1</td>
          </tr>
      </tbody>
  </table>
  <!-- 출력 -->
  
  ```
