# CSS

스타일시트

- css로 작성된 코드
- `<style>..<style>`

중요선택자 종류

| 종류          | 형태            | 사용예        |
| ------------- | --------------- | ------------- |
| 태그 선택자   | 태그            | h1            |
| 아이디 선택자 | #아이디         | #id           |
| 클래스 선택자 | .클래스         | .header       |
| 자손 선택자   | 선택자 > 선택자 | header > h1   |
| 후손 선택자   | 선택자 선택자   | header h1     |
| 링크 선택자   | 선택자 : link   | 선택자 : link |
